# ΠΧ8. Αξιολόγηση συμμετοχής μαθητή σε δραστηριότητα

**Πρωτεύων Actor**: Εκπαιδευτικός

**Ενδιαφερόμενοι**:  
**Εκπαιδευτικός**: θέλει να μπορεί να αξιολογήσει με μετρικές που υπάρχουν σε φόρμες, την επίδοση των μαθητών ανά χρονικές περιόδους.  

**Προϋποθέσεις**: Να υπάρχουν εγγεγραμμένοι εκπαιδευτικοί, μαθητές και δραστηριότητες στην πλατφόρμα.

## Βασική Ροή

### Α) Αξιολόγηση συμμετοχής μαθητή δραστηριότητα - μάθημα

1. Ο εκπαιδευτικός μπορεί αξιολογεί με δυναμικές μετρικές την επίδοση των μαθητών.
